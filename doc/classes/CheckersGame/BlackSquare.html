Class CheckersGame::BlackSquare
In: checkersGame.rb
Parent: Object

Stores the state of the black (playable) checkers squares. Attribute color is either CheckersGame::BlackSquare::BLACK_, CheckersGame::BlackSquare::RED or CheckersGame::BlackSquare::BLANK. Attribute kinged is either true or false depending on whether or not the piece in the current square is kinged.
